Анализ рисков
	Риск - вероятность появления каких-либо негативных факторов
	
	Типы рисков:
		1) Риски для проекта - вляют на график / ресурсы
		2) Риски для разрабатываемого продукта
		3) ...
		
	Текучесть разработчиков - риск для проекта
	Изменение в управлении - риск для проекта
	Неготовность аппаратных средств - риск для проекта
	Изменение требований - риск для проекта и для разрабатываемого продукта
	Задержка в разработке спецификации - риск для проекта и для разрабатываемого продукта
	Недооценка размеров разрабатываемой системы - риск для проекта и продукта
	Недостаточность средств - риск для продукта
	Изменение технологий в разработке ПО - бизнес-риск
	Появление конкурирующего программного продукта - бизнес-риск
