Модели решения функциональных и вычислительных задач
	Понятие модели и моделирования
		Модель - искуственно созданный объект, который даёт упрощённое представление о реальном объекте/процессе/явлении, отражающий существенные стороны изучемого объекта с точки зрения цели моделирования
		Модель не является абсолютной копией своего оригинала, отражает наиболее важные качества и свойства оригинала (те свойства, которые наиболее существенны для выбранной цели исследования)
		Модель - целевой образ объекта оригинала, который отражает свойства для достижения поставленной цели
		Модель - формализованное представление/описание оригинала, которое позволяет замещать оригинал для его лучшего изучения, исследования, описания его свойств
	
	Моделировние(эксперимент) - построение модели
	
	Направления
		1) Вычислительное - прямые рассчёты по созданной программе
		2) Инструментальное - испольщование базы знаний для построение программы
		3) Диалоговое - поддержание интерфейса пользователь <-> компьютер
	Цели создания моделей
		1) Познавательные
		2) Образовательные
		3) Управленческие
		4) Эксперементальные
		5) Созидательные
		
	Основные свойства моделей
		1) Эффективность - насколько правильно было использовать эту модель для достижения цели
		2) Универсальность
		3) Устойчивость - правильная работа модели в изменяющихся внешних условиях и экстренных ситуациях
		4) Содержательность - количество функций модели
			а) описательная
			б) интерпретаторская
			в) объяснительная
			г) предсказательная
			д) измерительная
		5) Ограниченность
		6) Полнота - Наличие сведений о объекте-оригинале, дя достижения поставленной цели
		7) Динамичность - Изменение модели с течением времени
		
	Виды моделей
		1) По области использования
			а) учебные
			б) обытные
			в) игровые
			г) имитационные
			д) научно-исследовательские
		2) По области знаний (деятелности человека)
			а) биологические
			б) экономические
			в) социологические
			г) исторические
			д) социологические
			и т.д.
		3) Временной фактор
			а) статические(структурная) - оражает строение и параметры объекта
			б) динамические - отражает процесс изменения объекта
		4) Форма представления модели
			а) математические
			б) геометрические
			в) словесные
			г) логические
		5) Способ представления
			а) материальные
			б) нематериальные(абстрактные, информационные) - в основе лежит информация (теория)
		
	Информационные модели
		Типы
			1) Табличные - объекты и войства педставлены в виде списка
			2) Иерархические
			3) Сетевые - для отражения систем, где связи между эдементами имеют сложную структуру
